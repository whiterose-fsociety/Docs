Welcome to Coding With Kids’s documentation!

Coding with kids can be a great way to not only teach themsome skills that will help them with their own lives,but also create or strengthen some lasting bonds between you and them.

Throughout your journey you will both likely learn new concepts and terms that will expand your horizons and provide hours of enjoyment

Caution

Some kids have problems with screen time and if you leave them unattended with a screen and they have access to things like YouTube

There are lots of great options for where to start, and Just Logic is a great place to start. Note that best starting point is another way to refer to the same place.

For details about the hardware and code options, see Options section for hardware and code.
